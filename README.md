# 独立游戏开发模拟器

一个像素艺术风格的文字决策类小游戏，沉浸式体验独立游戏开发者的艰辛历程。

## 游戏背景

您将扮演一位独立游戏工作室的创始人，带领3名小伙伴（美术师、程序员、策划师）在充满复古魅力的工作室中开发一款游戏。在有限的资金和时间内，您需要对开发过程中遇到的各种冲突和挑战做出明智的决策。

## 游戏机制

### 初始状态
- **启动资金**: 500万元
- **开发进度**: 0%
- **开始时间**: 2025年6月
- **团队成员信心**: 全员100%

### 核心玩法
- 点击「开始开发」或「继续开发」按钮推进时间
- 每个月都会遭遇一个随机冲突事件
- 每个事件提供2个选择，需要根据描述做出判断
- 选择会影响项目进度、剩余资金和团队成员信心

### 胜利条件
- 开发进度达到100%

### 失败条件
- 剩余资金降至0
- 任何团队成员信心降至0%

## 事件类型

游戏包含5种类型的冲突事件：

1. **外部环境冲突** - 竞争对手、投资人、合作伙伴等外部因素
2. **游戏设计取舍** - 核心玩法、美术风格、难度设计等关键决策
3. **团队协作问题** - 美术与程序、策划与程序等成员间的合作分歧
4. **团队意见冲突** - 不同成员对游戏方向的理念冲突
5. **个人问题** - 团队成员的私人困扰和职业焦虑

## 游戏特色

- **像素艺术风格**: 复古游戏工作室背景，沉浸式视觉体验
- **移动端优化**: 专为手机和平板设计的响应式界面
- **真实体验**: 基于真实独立游戏开发场景设计的事件
- **策略深度**: 每个选择都有利有弊，需要权衡考量
- **随机性**: 随机事件确保每次游戏体验不同
- **视觉反馈**: 清晰的状态显示和团队信心指示器

## 界面设计

- **顶部状态栏**: 金黄色资金显示，绿色进度显示
- **团队面板**: 左侧显示所有团队成员的实时信心状态
- **日期显示**: 右上角显示当前开发时间
- **事件面板**: 居中弹出的白色面板，清晰显示冲突事件
- **像素字体**: 使用等宽字体营造复古游戏氛围

## 如何开始

1. 直接在浏览器中打开 `index.html` 文件
2. 点击「开始开发」按钮开始游戏
3. 阅读事件描述，仔细选择应对策略
4. 观察选择后果，调整后续决策
5. 努力在资金耗尽或团队崩溃前完成游戏开发

## 技术实现

- **纯前端实现**: HTML + CSS + JavaScript
- **像素艺术**: 高质量像素风格背景图片
- **响应式设计**: 适配各种屏幕尺寸
- **复古UI**: 像素完美的按钮和面板设计
- **平滑动画**: CSS过渡效果提升用户体验
- **移动端优化**: 专门为触摸设备优化的交互
